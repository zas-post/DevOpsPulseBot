# 🏷 DevOps Pulse Bot (RSS Агрегатор)

Асинхронный Telegram-бот на **aiogram 3**, работающий по архитектуре очередей (Продюсер / Консьюмер). Бот автоматически опрашивает заданные RSS-ленты (Хабр, VC, Tproger, DZone, Dev.to), складывает новые посты в локальную базу данных SQLite и плавно выводит их администратору на модерацию, полностью исключая риск блокировок со стороны Telegram API (`Flood control`).

---

## 🚀 Архитектура системы

Проект разделен на два независимых контура, работающих параллельно:
1. **Продюсер (Producer):** Запускается по расписанию (раз в 15 минут) через `APScheduler`. Быстро скачивает все новые посты из RSS-лент и сохраняет их в базу данных со статусом `PENDING`. Не отправляет сообщения в Telegram.
2. **Консьюмер (Consumer):** Работает как бесконечный воркер (`asyncio.create_task`). Он методично забирает из базы по одному посту и отправляет админу на модерацию строго с задержкой в **3 секунды**.

---

## 🛠 Технологический стек

- **Язык:** Python 3.11+
- **Фреймворк бота:** aiogram 3.x (Асинхронный)
- **Планировщик:** APScheduler 3.x
- **База данных:** SQLite + SQLAlchemy 2.0 (Асинхронный драйвер `aiosqlite`)
- **Парсинг и HTTP:** feedparser + httpx
- **Контейнеризация:** Docker, Docker Compose
